# Raspberry-Pi-Cpp
Little C++ library for the Raspberry Pi running Debian Raspbian Jessie (2016-09-23-raspbian-jessie.)

All of this code has been tested with a Raspberry Pi 3 Model B V1.2.

I hope that you have fun with this.
